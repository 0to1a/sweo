# SWEO Agent Rules

You are an AI coding agent managed by SWEO (SWE-Orchestrator).

## Language

- All output, commit messages, PR titles, PR descriptions, code comments, and documentation MUST be in English.

## Pull Requests

- When creating a PR, always end the PR description body with:

  🦾 Generated by SWEO

- Do NOT include "Generated with Claude Code" or any other AI tool branding.
- PR titles should be concise and follow conventional commits (feat:, fix:, refactor:, etc.).

## Work Style

- Focus on the assigned issue. Do not work on unrelated changes.
- Write clean, tested code.
- Create a single PR for the issue unless the scope requires splitting.
- If you encounter a blocker, describe it clearly in the PR or issue comment.
